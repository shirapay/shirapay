# **App Name**: ShiraPay

## Core Features:

- Vendor Onboarding & Verification: Secure Vendor signup with profile verification, managed by admin approval via Firestore.
- Invoice Generation (Vendor): Generate unique, trackable invoices with QR codes and shareable links. Transaction ID generator with data logging and potential use as tool with AI suggestion feature.
- Real-Time Transaction Tracking (Vendor): Display immediate feedback via success animations upon payment confirmation.
- Invoice Scanning (Agent): Scan QR codes to fetch invoice details and automatically route for approval to different departments based on AI suggestions and validation tool to avoid potential routing errors.
- Admin Approval Center: Display real-time list of pending transactions with approve/reject actions.
- Spending Analytics Dashboard: Interactive analytics dashboard to filter/audit spending by agent, vendor, status and date range.
- Firestore Integration: Secure storage and retrieval of user and transaction data.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey trust and security.
- Background color: Light Gray (#E8EAF6) for a clean and professional interface.
- Accent color: Vivid Orange (#FF5722) to highlight important actions and notifications.
- Font pairing: 'Inter' (sans-serif) for body text and headlines.
- Use minimalist, professional icons that are easily understandable.
- Mobile-first, responsive design adapting to different screen sizes.
- Subtle animations for transitions and status updates to enhance user experience.
